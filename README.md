# YouTube Video to Local Markdown Summary

Скрипт на Python для автоматического извлечения субтитров из видео на YouTube и генерации структурированных учебных/технических конспектов в формате Markdown. Процесс суммаризации полностью конфиденциален и автономен, так как выполняется на локальном сервере **`llama.cpp`**.

Скрипт оптимизирован для работы в условиях сетевых ограничений (поддерживает работу через прокси и авторизацию через cookies), автоматически очищает ссылки от лишних параметров плейлистов и защищен от зацикливания локальных моделей при генерации больших текстов.

## 🛠️ Системные требования и зависимости

1. **Python 3.8 или выше**
2. **Локально запущенный сервер `llama.cpp`** (в режиме `llama-server.exe`) с моделью класса Instruct (например, *Meta-Llama-3.1-8B-Instruct*).
3. **Сторонние библиотеки Python:**
   - `yt-dlp` — для стабильного извлечения текстовых дорожек с YouTube.
   - `openai` — официальный клиент для удобной отправки запросов на совместимый API-эндпоинт сервера `llama.cpp`.
