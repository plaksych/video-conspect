# Обзор программы Obsidian и метода ZLE Custom

## Введение

Obsidian - это программа для заметок в формате Mark, которая позволяет хранить заметки на девайсе без облака и риска потерять доступ. Метод ZLE Custom - это система для ведения заметок и управления знаниями, созданная немецким социологом Николасом Луманом.

## Основные принципы ZLE Custom

### Типы заметок

- **Мимолётные заметки**: быстрые временные мысли.
- **Литературные заметки**: краткое изложение мыслей автора.
- **Постоянные заметки**: проработанные вечные идеи.

## Плагины Obsidian

- **Advanced Tables**: плагин для создания таблиц и манипулирования ими.
- **Dataaview**: плагин для работы с заметками Obsidian как с базой данных.

## Возможности Obsidian

- **Создание шаблонов**: позволяет сохранять и комбинировать идеи.
- **Синхронизация заметок**: позволяет синхронизировать заметки с OneDrive и хранить их на компьютере.

## Выбор инструмента для заметок

- **Важно выбирать инструмент**, который решает конкретные проблемы и задачи.
- **Необходимо найти баланс** между функциональностью и простотой инструмента.
- **Если инструмент хорошо работает**, нет необходимости в миграции на другой продукт.

## Термины

- **Obsidian**: инструмент для заметок.
- **ZLE Custom**: метод ведения заметок и управления знаниями.
- **Мимолётные заметки**: быстрые временные мысли.
- **Литературные заметки**: краткое изложение мыслей автора.
- **Постоянные заметки**: проработанные вечные идеи.
- **Advanced Tables**: плагин для создания таблиц.
- **Dataaview**: плагин для работы с заметками как с базой данных.
- **OneDrive**: облачное хранилище, которое позволяет синхронизировать заметки с компьютером.
